## Stack


A Go implementation of a stack data structure. It accepts any type of value.

![codecov.io Code Coverage](https://img.shields.io/codecov/c/github/dwyl/hapi-auth-jwt2.svg?maxAge=2592000)

## Usage

```go
stack.push(1)
stack.push(true)
stack.push("hello")

stack.pop()
stack.pop()
stack.pop()
```
